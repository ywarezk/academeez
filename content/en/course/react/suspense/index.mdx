In these series of lessons we will cover the important things we should know about React `<Suspense>`.

## What is React Suspense?

`<Suspense>` allows us to suspend rendering while we are waiting for something to load asynchronously. It is a component that lets you specify the loading indicator or fallback content while waiting for something to load.
