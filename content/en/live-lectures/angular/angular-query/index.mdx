In this lecture we will talk about bad practice that is common when using NGRX state management.  
TLDR; Please stop using `NGRX` for storing server query cache.

## When

The lecture will be on Tuesday 19/12/2023 at 18:00 (Israel time)

## Topics

We will discuss the following topics:
- NGRX state management
- `@ngrx/entity`
- `@ngrx/data`
- `@ngrx/effects`
- `@ngrx/store`
- `@ngrx/store-devtools`
- angular query `@ngneat/query`

## Prerequisites

- Experience with Angular
- Experience with NGRX
