There are times where you need to change the `font-size` of a text according to the size of the screen.  
For example you have a very big display heading which needs to be smaller on smartphones in order to fit the screen size.  
You have an article and you want to make the font bigger on smartphones so the users will be able to read the content on a small screen.  
In this lesson we will learn about responsive typography, for cases where the `font-size` needs to change according to the screen size.
