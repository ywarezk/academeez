# Prerequisites

These are few things you will need to know before starting this course

## HTML

Basic knowledge of HTML.  
Being more specific you should know how to create the following using HTML

- html structure
- head-title-meta tags
- body tag
- creating a navigation menu
- creating lists
- headers-paragraph tags
- anchor links

## CSS

Basic knowledge of CSS.  
Including the following.  

- position
- display
- flexbox
- height-width
- color, background-color

## Javascript

We will be using Javascript as our programming language throughout this course.  
You need to know the following:

- Variable declaration
- ifs
- loops
- functions
- lambda functions
- oop
- destructuring
- spread operator
