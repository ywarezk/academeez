## Falsy value

A Falsy value is a value that Javascript will evaluate to `false`.  
This means that if this value will be placed in an `if statement` Javascript will consider them as false.  

The values are: `undefined`, `null`, `NaN`, `0`, `""`, `false`

All other values if placed in condition, Javascript will consider as `true`.  
For exmaple an empty object `{}` or empty array `[]` are considered as truthy.
