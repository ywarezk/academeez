If we do learn React, what kind of app can we build with React

## Apps built with React

We can build the following

- Website - for example the Netflix website is built with React
- Native mobile apps - For example the facebook and instagram apps are built with React Native
- Desktop apps - for Windows, Linux, MacOs
- Hybrid Mobile app

How is it possible for React to serve so many different frontend apps.

## Platform Agnostic

Platform is whomever is running our application:

- iOS
- Android
- Desktop
- Browser
- Node.js

React is platform agnostic meaning it can run on different platforms that can run Javascript.

## React by itself does not draw the UI

## Different apps we can build with React

Using React we can build:

- Native Mobile apps
- Hybrid Mobile apps
- Desktop applications
- WebSites

## Native Mobile apps

To build native mobile apps using React, we will have to combine `React` with `react-native` which allows us to use React to develop native mobile screen to **iOS** and **Android**

So we will have to learn the core of React along with specific `react-native` platform and components.

## Hybrid App

These type of apps represent mobile apps as well that can run on **iOS** and **Android**.  
The approach is somewhat different, in these apps the mobile app will open a browser and React will run in that browser to draw the app.  
To create this you will have to combine `React` with `react-dom` to make react draw the app in the browser environment, and also learn how to interact with the platform using `Cordova` or `Ionic`

## Desktop Apps

We can use React to create desktop applications that can run **Windows**, **MacOS** and **Linux**.  
To build those apps with React we will have to use the `Electron` library, which works similar to **Hybrid apps** and open a browser as a desktop application.  
To build those apps we will have to combine the `React` library with `react-dom` and learn the `Electron` platform to communicate with the OS.

## Website

Using React we can create websites that run on modern web browsers.  
To build those apps we will have to combine `React` with `react-dom` and work with the browser platform environment.

## Summary

This course will focus on creating a website using `React`.  
We will combine `React` with a library called `react-dom`.  
We will work on the browser platform and browser api
