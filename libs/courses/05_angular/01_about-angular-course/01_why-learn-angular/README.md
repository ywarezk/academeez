Let's go over few reasons why you should learn Angular and take this angular course.

## Build frontend apps

One framework to rule them all.  
Angular is used mainly to create fast web sites, but it can also be used to create apps for different frontend platforms:

- Hybrid smartphone apps with [Ionic](https://ionicframework.com/)
- Native smart phone apps with [NativeScript](https://nativescript.org/)
- Desktop apps using [Electron](https://www.electronjs.org/)
- Fast single page application web sites.

## Open source project managed by Google

Angular is an open source project managed by google.  
The project has many contributers and is very popular among the open source project.  
The fact that google is managing the project and the organization is creating their application using the technology, gives us comfort since it reduce the change that this technology will be abandoned in the future.  
It's MIT license which means you can use it commercially.

## Written with Typescript

Angular is written using [Typescript](https://academeez.com/courses/typescript) which means perfect Typescript support, with all the benefits that typescript brings to the table

## Framework

Angular provides an entire infastructure which points us to the right direction in our development.  
It gives us tools for dealing with forms, routing, dependency injection, sharing data, and more.  
Working with a framework points us to the best development practices.

## Popularity

Angular was formaly released at the end of 2016, and since then became one of the most popular tools for creating single page application, 2nd to [React](https://academeez.com/courses/react) the popularity of angular has increased the past years.  
Combining with the fact that it is an open source project with large community which encourage the creation of libraries, this created a lot of quality libraries which you can use to accelerate your project development.  
The popularity can also be seen in work places which are looking to hire angular developer.  
No doubt that you will drastically improve your market value, if your knoledgebase will include angular experience.
