As I often say, there is often more than one good solution for a certain problem.  
Having said that to solve your data storage need, you will have various db technologies you can choose from, and a lot of them are good.  
The variety of choices can often leave the beginning developer overwhelmed, so we will try to help you navigate from all the choices out there.
