Most of the servers you are going to create will be connected to a database.  
Databases help us store information and query that information. We are going to often use it to store data about our users, and display customize data for those users.  
The data in our database is persistent, and even if our user returns tomorrow to our app, we can still send him his previous data and information.  

Backend developers deal a lot with databases, that is why it's really important to master this skill if you want to be a good backend developer.
