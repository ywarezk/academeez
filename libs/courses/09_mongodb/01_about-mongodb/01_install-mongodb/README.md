## Community

We will work with the community free version.  
The community version is open sourced and free to use, while for the enterprise version you will have to pay.

## Windows installation

download the installer [here](https://www.mongodb.com/try/download/community)  
MongoDB will store it's file in a directory you will have to create.  
By default the directory is `C:\data\db`.  
Create that folder.

You can run the mongodb server with the following command in the CMD:

```bash
> C:\mongodb\bin\mongod.exe
```

This will run the server an now you can run the shell to verify that you are able to connect to your database.

```bash
> C:\mongodb\bin\mongo.exe
```
