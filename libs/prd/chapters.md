# Chapters page

When a user clicks to learn a course he is being directed to the url `/courses/:courseName`  
In here there is an ordered list of topics.
There is a big call for action `Start Learning` or `Continue Learning` if he already started this course.  

There is an ordered list of chapters.  
clicking an chapter will usually learn to [Lesson Page](lesson.md)

But a chapter can also have subchapters (A tree like structure) and in that case the user will see another chapter page.  

Each chapter will have a description in a `Chapter Card`

## Chapter Card

Chapter card will contain
- title
- description
- share
- play button for playing the first video of the chapter
- Call for action to start learning the chapter

## Prerequisites

There is a section on the right side of the screen which shows a list of `Prerequisites` needed for the course.

Each `Prerequisit` is similar to a course card in the homepage
