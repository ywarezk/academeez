# Chapters page

When a user clicks to learn a course he is being directed to the url `/courses/react`  
In here there is an ordered list of topics.
There is a big call for action `Start Learning` or `Continue Learning` if he already started this course.  

There is an ordered list of chapters.  
clicking an chapter will usually learn to [Lesson Page](lesson.md)

But a chapter can also have subchapters (A tree like structure) and in that case the user will see another chapter page.  

Each chapter will have a description in a `Chapter Card`
