# Academeez layout

The layout will be displayed in all the pages and will contain a `Header` and a `Footer`

## Header

1. `Logo`

Will be placed on the left side of the Header.  
The line of the color will be in `svg` and should change color according to course the user is in.  
Note that a course has a primary logo color and we are adding that color to certain elements according to where the user is in.

2. `Navigation`

The header will contain a right side navigation.  
For the first version there will be a link to the following pages:
- [courses](courses.md) - This is the call for action and should probably stand out a bit above the others
- blog
- donate
- github icon - leading to the open source project
- youtube icon - leading to the lectures on the channel in youtube

All the links behaviour should follow [style guide](style-guide.md)

3. `Search`

The user will be able to search for a lesson of his choice.  
Searching will lead to the [Search Page](search-page.md)

4. `Footer`

- The footer will create links to the different sections of the app.  
- copyright text
- created by Nerdeez text

## Responsive

The header will change for small devices and the links will be displayed in an hamburger menu
