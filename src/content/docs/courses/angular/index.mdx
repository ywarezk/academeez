[Angular](https://angular.dev) is a framework for building single-page web applications using TypeScript.  
In this course we will learn how to use Angular to create a web application, we will interact with server, and learn about the popular libraries and frameworks that are used with Angular, such as RxJS, Angular Material, and NgRx.
